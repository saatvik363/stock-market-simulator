
# Stock Market Simulator (Paper Trading)

## Overview
This is a web-based stock market simulator built for students to learn
how stock trading works using real market prices and fake money.

The platform uses paper trading, meaning no real money is involved.

## Marketplace Type
- Financial marketplace
- Stock market (paper trading)

## Asset Universe
- US Stocks (NYSE / NASDAQ)

## Wallet Rules
- Initial balance: $100,000 (virtual USD)
- Currency: USD
- No real money involved

## Trading Rules
- Market orders only
- Real market prices via API
- User trades do NOT affect prices
- No short selling
- No margin or leverage

## Tech Stack
- Backend: Node.js + Express
- Database: MongoDB
- Frontend: To be decided
